# CrimeScene

Analyze your codes. This gem provides:

- Constant references from package to package

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'crime_scene'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install crime_scene
